### 1. User Registration
1. Build a frontend form with fields for name, email, phone, address, password, confirm password, and client-side validation.  
2. Create backend endpoints to validate input, check duplicates, hash passwords, and store encrypted sensitive fields.  
3. Send verification email with token and handle confirmation flow; return clear error messages.  
4. **Importance:** High — foundational for all users.  
5. **When to complete:** Sprint 1 (Weeks 1–2).  
6. **Benefit:** Lets people create accounts and start using the platform.
